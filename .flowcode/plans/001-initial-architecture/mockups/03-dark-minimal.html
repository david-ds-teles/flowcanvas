Phase 2 — Canvas Shell

Mount React Flow inside a client-only shell. The adapter maps the JSONCanvas document onto { nodes, edges }; selection and dirty state live in the Zustand store.

typescript · adapter.ts
// FlowcanvasDoc → React Flow {nodes,edges}
export function toReactFlow(doc: FlowcanvasDoc) {
  const nodes = doc.nodes.map(adaptNode)
  const edges = reconcileEdges(
    deriveLinkEdges(doc.nodes),
    doc.edges,
  )
  return { nodes, edges }
}

Derived links: edges get deterministic ids (lk:<from>-><to>) so every load replaces the whole derived set without touching user or agent edges.

Acceptance — board renders from ?path; minimap + controls visible; no SSR error (ssr:false).
