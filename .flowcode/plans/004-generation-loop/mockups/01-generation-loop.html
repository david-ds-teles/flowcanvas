The board is a decomposed view of this document. Every component traces back to a heading here via meta.source; the typed derives-from edges trace the extraction.

↔ linked to “Auth Service” on canvas

Auth Service

Issues and verifies session tokens for every request. The only node permitted to mint a token; everything else calls verify(token). The API Gateway informs it on each request.

Order Service

Owns the order lifecycle and POST /orders; produces order.created onto the Event Bus and calls Payment.

Orders DB

Postgres datastore for orders + line items. Written only by the Order Service.
